## 执行计划

### Phase 1: 基础数据层 (预计 2h)
**任务**：T2 灵石数据  
**输出**：
- `src/data/stone-chain.js` - 10级灵石数据
- `src/data/realms.js` - 9境界数据
- `src/data/elements.js` - 五行数据
- `src/data/shop-items.js` - 商店商品数据

### Phase 2: 核心算法 (预计 4h)
**任务**：T1 合成算法  
**依赖**：Phase 1  
**输出**：
- `src/core/merge-engine.js` - BFS连通块+合成判定
- `src/core/__tests__/merge-engine.test.js` - 单元测试

### Phase 3: 游戏系统 (预计 6h)
**任务**：T3 修炼突破 + T4 工坊商店  
**依赖**：Phase 1, Phase 2  
**输出**：
- `src/systems/cultivation.js` - 修炼/突破逻辑
- `src/systems/shop.js` - 商店系统
- `src/systems/state.js` - 全局状态管理

### Phase 4: 动画效果 (预计 6h)
**任务**：T5 合成动画 + T6 突破动画  
**依赖**：Phase 3（需要游戏状态驱动）  
**输出**：
- `src/animations/merge.css` - 合成动画样式
- `src/animations/breakthrough.css` - 突破动画样式
- `src/animations/controller.js` - 动画控制器

### Phase 5: 集成调试 (预计 6h)
**任务**：T7 主循环 + T8 调试工具  
**依赖**：全部  
**输出**：
- `src/game.js` - 游戏主循环
- `src/tools/debug.js` - 调试工具
- `index-v6.0.html` - 完整游戏页面
