## 三、核心玩法设计

### 3.1 合成规则

#### 基础合成（相邻3+相同）
```
相邻判定：上下左右4方向
合成条件：3个或以上相同元素相邻
合成结果：合并为下一级灵石

灵气(0级) ×3 → 灵石Lv1
灵石Lv1 ×3 → 灵石Lv2
灵石Lv2 ×3 → 灵石Lv3
...
灵石Lv9 ×3 → 仙晶（最高级）
```

#### 五行相生（加成机制）
```
相生链：木 → 火 → 土 → 金 → 水 → 木
当合成包含相生元素时：
- 纯度高10%以上视为「完美合成」
- 完美合成产出等级+1

示例：
木灵气(90%) + 木灵气(85%) + 木灵气(95%) 
+ 相邻火灵气(80%)
= 灵石Lv2（普通为Lv1）
```

#### 自动合成（工坊解锁）
- 购买「自动合成仪」后开启
- 每次捕捉后自动执行一次最优合成
- 不消耗手动操作次数

### 3.4 图鉴系统

收集要素：
- **灵石图鉴**：首次合成出某等级灵石时解锁
- **五行图鉴**：捕捉/合成过某元素N次解锁
- **完美图鉴**：完成完美合成时解锁

收集奖励：
- 每解锁一个图鉴：修为上限+1%
- 集齐一套五行：捕捉纯度+2%
- 集齐所有灵石：获得称号「合成大师」

### 3.6 炼丹系统

丹药配方：
```
聚气丹：任意3个Lv1灵石 → 突破时修为+20%
清心丹：金+木+水 → 捕捉纯度+10%持续10次
烈火丹：火+火+火 → 下次合成必为完美
土灵丹：土+土+土 → 下次捕捉必为土元素
混元丹：5种元素各1 → 随机获得1个Lv3灵石
```

## 六、设计理念

> **「简单上手，深度钻研」**

1. **零学习成本**：点击即玩，无需教程
2. **渐进解锁**：随着境界提升逐步开放功能
3. **策略深度**：五行相生、纯度管理、空间规划
4. **成就感循环**：合成→升级→突破→新能力→更高级合成
