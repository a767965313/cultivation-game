# 修仙合成系统 - 需求文档

## 核心玩法概述
穿越修仙世界，通过捕捉天地灵气，在8x8合成器中提纯炼制五行灵石，不断提升修为。

## 核心机制

### 1. 灵气获取
- 点击"捕捉灵气"按钮
- 随机获得1单位**驳杂灵气**（属性随机：金/木/水/火/土）
- 纯度：10%（基础值）

### 2. 合成器系统
- 8×8方格，共64格
- 每格可放置1个单位物品
- 物品类型：驳杂灵气 → 灵石（1-5级）

### 3. 合成规则

#### 基础合成（3连消）
- **条件**：3个连续（横向或纵向）的同阶同级同属性物品
- **结果**：合成1个高阶物品
- **位置**：存放到最后一次放置的位置
- **纯度提升**：
  - 3个驳杂灵气（10%）→ 1个精纯灵气（30%）
  - 3个30% → 1个60%
  - 3个60% → 1个90%
  - 3个90% → 1个100%灵石

#### 高级合成（十字架）
- **条件**：9个相同物品呈十字架排列
  ```
    X
    X
  X X O X X  （O为最后放置位置）
    X
    X
  ```
- **结果**：合成1个高阶物品，品质大幅提升
- **加成**：十字架合成有额外灵气加成（1.5倍）

### 4. 灵石系统
- 当灵气达到100%时自动转化为**灵石**
- 五行属性：金、木、水、火、土
- 灵石等级：1级 → 2级 → 3级 → 4级 → 5级（仙晶）
- 3个同级灵石 → 1个高1级灵石

### 5. 修为系统（MVP后扩展）
- 吸收灵石提升修为
- 修为等级：炼气 → 筑基 → 金丹 → 元婴 → 化神...

## MVP版本功能
1. 8×8合成器界面
2. 点击获取驳杂灵气
3. 拖拽/点击放置物品
4. 3连合成检测与动画
5. 十字架合成检测
6. 灵气纯度显示
7. 灵石生成与等级显示
8. 基础UI（灵气库存、合成器、操作按钮）

## 扩展规划（后续版本）

### V1.1 修为系统
- 灵石可转化为修为
- 境界突破系统
- 境界影响灵气获取效率

### V1.2 功法系统
- 不同功法影响合成效率
- 功法升级
- 特殊合成配方

### V1.3 自动化
- 灵气自动捕捉（低效率）
- 自动合成辅助
- 挂机收益

### V1.4 探索系统
- 不同地图有不同灵气属性偏向
- 探索获得稀有材料
- 秘境副本

### V1.5 炼丹/炼器
- 灵石用于炼丹
- 丹药提供临时buff
- 炼器装备提升属性

## 技术实现
- 前端：HTML5 + Canvas 或 DOM
- 游戏循环：requestAnimationFrame
- 数据结构：二维数组表示合成器网格
- 合成检测：滑动窗口算法检测3连/十字架

## 数据结构示例
```javascript
// 网格单元格
{
  type: 'qi',      // 'qi' | 'stone'
  element: 'fire', // 'gold' | 'wood' | 'water' | 'fire' | 'earth'
  purity: 10,      // 10, 30, 60, 90, 100
  level: 0,        // 0=驳杂灵气, 1-5=灵石等级
  id: 'uuid'
}
```
