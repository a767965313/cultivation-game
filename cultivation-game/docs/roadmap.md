# 修仙合成系统 - 游戏开发路线图

## 🎯 MVP版本（已完成）

### 核心功能
- ✅ 8×8合成器网格
- ✅ 点击捕捉灵气（随机属性）
- ✅ 物品放置与收回
- ✅ 3连合成检测（横向/纵向）
- ✅ 十字架合成检测（9格）
- ✅ 灵气提纯系统（10%→30%→60%→90%→100%）
- ✅ 灵石生成与等级系统
- ✅ 基础UI与动画效果

---

## 📅 版本规划

### V1.1 修为系统
**目标**：增加养成维度，让玩家有长期目标

#### 功能清单
- [ ] **境界系统**
  - 炼气期 → 筑基期 → 金丹期 → 元婴期 → 化神期 → 炼虚期 → 合体期 → 大乘期 → 渡劫期
  - 每个境界分前期/中期/后期/大圆满

- [ ] **修为吸收**
  - 点击灵石吸收转化为修为
  - 不同属性灵石提供不同修为值
  - 高等级灵石提供指数级修为

- [ ] **境界突破**
  - 修为满后需要特定条件突破
  - 突破可能需要特定组合的灵石
  - 突破失败惩罚机制

- [ ] **境界加成**
  - 境界影响灵气获取效率
  - 境界影响合成成功率
  - 境界解锁新功能

---

### V1.2 功法系统
**目标**：增加策略性，不同流派玩法

#### 功能清单
- [ ] **功法获取**
  - 通过特定合成解锁功法残页
  - 收集残页合成完整功法

- [ ] **功法效果**
  | 功法名称 | 效果 |
  |----------|------|
  | 金灵诀 | 金属性灵气+20%纯度 |
  | 木灵诀 | 木属性合成有概率额外产出 |
  | 水灵诀 | 水属性可流动到空位 |
  | 火灵诀 | 火属性合成时提升周围 |
  | 土灵诀 | 土属性更稳定，不易被合成 |

- [ ] **功法升级**
  - 消耗同属性灵石升级功法
  - 功法等级影响效果强度

---

### V1.3 自动化系统
**目标**：减少重复操作，增加挂机收益

#### 功能清单
- [ ] **自动捕捉**
  - 消耗灵气开启自动捕捉
  - 效率低于手动，但可持续

- [ ] **自动合成**
  - 设置自动合成规则
  - 满足条件自动执行

- [ ] **阵法系统**
  - 放置阵法在合成器边缘
  - 阵法提供各种自动化效果

---

### V1.4 探索系统
**目标**：增加外部交互，获取稀有资源

#### 功能清单
- [ ] **地图探索**
  - 不同区域有不同灵气属性偏向
  - 金灵山脉、木灵森林、水灵海域等

- [ ] **探索收益**
  - 发现稀有灵气（双属性、变异属性）
  - 获取功法残页
  - 遇到灵兽战斗

- [ ] **秘境副本**
  - 限时挑战关卡
  - 特殊合成规则
  - 丰厚奖励

---

### V1.5 炼丹炼器系统
**目标**：增加物品消耗途径，丰富玩法

#### 功能清单
- [ ] **炼丹系统**
  - 五行灵石按配方炼制丹药
  - 丹药提供临时buff（自动合成、纯度加成等）

- [ ] **炼器系统**
  - 炼制法宝装备
  - 法宝提供被动效果
  - 法宝可升级强化

#### 丹药配方示例
| 丹药名称 | 配方 | 效果 |
|----------|------|------|
  | 灵气丹 | 任意3个1级灵石 | 自动捕捉1小时 |
  | 提纯丹 | 金+水+火各1个 | 下次合成纯度+20% |
  | 爆发丹 | 5个同属性1级灵石 | 下次合成数量×2 |

---

### V1.6 社交系统
