# V6.0 任务拆分与 Agent 分配

## 任务分配原则

1. **领域专业化**：每个任务分配给最擅长的 agent
2. **接口清晰**：模块间通过明确接口通信
3. **独立测试**：每个任务交付物可独立验证
4. **并行开发**：最大化并行效率

---

## 任务列表

### T1: 合成算法引擎
**领域**：算法 / 数据结构  
**难度**：⭐⭐⭐⭐  
**依赖**：无  
**交付物**：
- BFS连通块查找算法
- 合成触发判定
- 完美合成判定（纯度+相生）
- 单元测试（10+场景）

**接口定义**：
```typescript
function findConnected(grid: Grid, start: number, element: ElementType): number[]
function checkMerge(grid: Grid, index: number): MergeResult | null
function executeMerge(indices: number[]): { newItem: Item, isPerfect: boolean }
```

**验收标准**：
- [ ] 能正确识别3-8个连通元素
- [ ] 边界情况处理正确（角落/边缘）
- [ ] 完美合成判定准确
- [ ] 性能：1000次合成 < 100ms
