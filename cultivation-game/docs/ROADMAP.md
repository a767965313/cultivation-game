# 修仙合成系统 - 项目总路线图

## 项目愿景
打造一款**放置+合成+修仙**的轻量级手游，核心体验：
- 30秒上手，无限深度
- 每5分钟一次爽点（突破/合成高阶/解锁成就）
- 可离线成长，适合碎片时间

## 各版本详细规划

## V6.0 - 核心玩法版

### 核心交付标准
玩家可以：
1. 捕捉灵气并放置到网格
2. 3个相邻相同元素自动合成升级
3. 合成出的灵石可以吸收为修为
4. 修为满后突破境界
5. 用灵石货币购买工坊升级

### 功能清单
- [ ] 合成系统（BFS算法+完美合成判定）
- [ ] 灵石升级链（0-9级完整数据）
- [ ] 修炼系统（吸收转修为）
- [ ] 突破系统（9境界×4阶段）
- [ ] 工坊系统（4个基础商品）
- [ ] 基础动画（合成/突破）
