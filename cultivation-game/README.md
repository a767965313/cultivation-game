# 🔮 修仙合成系统

> 捕捉天地灵气，提纯五行灵石，突破境界，踏上仙途

一个基于 Web 的休闲合成类游戏，玩家通过捕捉灵气、合成提纯、修炼突破，体验修仙之路。

## 🎮 核心玩法

### 1. 捕捉灵气
- 点击按钮捕捉随机五行灵气（金⚔️、木🌿、水💧、火🔥、土🏔️）
- 灵气纯度初始 10%，可通过合成提升

### 2. 合成提纯
- 将灵气放入 8×8 合成器
- **严格相邻**（上下左右）的 3 个以上相同灵气可合成
- 合成数量越多，品质提升越高
- 纯度达到 100% 自动转化为灵石

### 3. 修炼突破
- 吸收灵石获得修为
- 突破九大境界：炼气→筑基→金丹→元婴→化神→炼虚→合体→大乘→渡劫
- 每境界分前期/中期/后期/大圆满
- 高境界提升灵气获取效率

### 4. 灵气工坊
- 用灵石/合成点购买道具
- 自动捕捉器、合成加速、修为增幅、聚灵符等

## 🎯 合成规则详解

### 相邻定义
只计算**严格相邻**（上下左右），不含对角线：
```
  ↑
← ● →
  ↓
```

### 品质提升
- 3个合成：纯度 ×3
- 4个合成：纯度 ×4
- 5个合成：纯度 ×5
- 5个以上：额外 +50% 纯度

### 合成示例
- (0,0), (0,1), (0,2) - 横向三连 ✓
- (1,1), (2,1), (3,1) - 纵向三连 ✓
- (0,0), (0,1), (1,1) - L形三连 ✗（不相邻）

## 💾 存档说明

游戏数据保存在浏览器本地存储（LocalStorage）中：
- 自动保存：每 30 秒自动存档
- 手动保存：点击"💾 保存"按钮
- 数据不丢失：只要不清理浏览器数据，存档一直保留

**⚠️ 注意**：更换浏览器或清理缓存会丢失存档！

**📤 跨设备同步**：使用菜单中的"导出存档"和"导入存档"功能

## 📱 PWA 安装

1. 打开游戏网页
2. 点击地址栏右侧"⋮"
3. 选择"添加到主屏幕"
4. 点击"安装"

### iOS (Safari)
1. 打开游戏网页
2. 点击底部分享按钮
3. 选择"添加到主屏幕"
4. 点击"添加"

### 桌面端 (Chrome/Edge)
1. 打开游戏网页
2. 地址栏右侧出现"⊕"图标
3. 点击"安装修仙合成系统"
