# Gitee Pages 部署指南

## 快速迁移

```bash
# 1. 在 gitee.com 创建同名仓库 cultivation-game

# 2. 添加 Gitee 远程仓库
git remote add gitee https://gitee.com/a767965313/cultivation-game.git

# 3. 推送代码
git push gitee main

# 4. 在 Gitee 仓库设置中开启 Pages 服务
# 路径：仓库 → 服务 → Gitee Pages
```

## 访问地址

部署完成后访问：
https://a767965313.gitee.io/cultivation-game/

## 注意事项

- Gitee Pages 需要实名认证
- 免费版有流量限制
- 国内访问速度优于 GitHub Pages
